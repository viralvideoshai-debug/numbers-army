SAVE YOUR 4 GENERATED IMAGES IN THIS FOLDER WITH EXACTLY THESE NAMES:

  icon.png    -> the royal blue app icon (crowned capsule character in gold frame)
  hero.png    -> the ice-canyon key art (blue army charging the pink castle)
  skins.png   -> the 6-character skin sheet (blue/ninja/wizard/gold/knight/crystal)
  bosses.png  -> the 3-boss lineup (red king / orange stomper / navy cannon king)

The game automatically uses icon.png (browser tab / PWA icon) and hero.png
(menu background) as soon as they exist. skins.png and bosses.png are
reference art for store screenshots and future skin models.

PNG format. icon.png should be 1024x1024 (it gets scaled down for the
Play Store, which needs exactly 512x512 - any image tool can resize it).
